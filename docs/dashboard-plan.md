# 당뇨 식단 대시보드 구상 문서

작성일: 2026-07-01
프로젝트 ID: P09
프로젝트명: mom-diabetes-dashboard

## 1. 문제 정의

어머니는 오랜 당뇨 병력이 있고 인슐린을 많이 사용 중이며, 2026-07-01 병원 검사에서 A1C가 11.4%로 확인됐다. ADA는 A1C가 최근 2-3개월 평균 혈당을 보여주며, 대부분 성인 당뇨 환자의 일반 목표가 7% 미만이라고 설명한다. A1C 11.4%는 의료진의 적극적인 치료 계획이 필요한 높은 상태다. 다만 이 프로젝트의 역할은 치료 관리가 아니라, 보호자가 식사 선택 순간에 참고할 수 있는 쉬운 음식/레시피 안내를 제공하는 것이다.

A1C 11.4%의 추정 평균혈당(eAG)은 약 280 mg/dL이다. 이는 공식 표의 10% = 240 mg/dL보다 높은 수준이며, 실제 목표와 약물 계획은 의료진이 개인 상태에 맞춰 정해야 한다.

## 2. 제품 목표

- 음식 앞에서 바로 검색해 "먹어도 되는지, 얼마나, 무엇과 같이 먹을지"를 확인한다.
- 좋아하는 한국 음식은 무조건 금지로만 다루지 않고 혈당 부담을 줄인 버전으로 바꾼다.
- 가족이 장보기와 식단 준비를 반복 가능하게 만든다.
- 복잡한 기록 없이 GitHub Pages에서 바로 열 수 있는 정적 대시보드로 공유한다.

## 3. 비목표

- 인슐린 용량 추천
- 진단 또는 치료 결정
- 인슐린 투여량 기록/분석
- 운동량 측정/처방
- 혈당 추세 분석 중심 대시보드
- 극단적 저탄수화물/금식 지시
- 합병증별 식단 처방. 신장질환, 심혈관질환, 저혈당 무감지 여부 확인 전까지 세부 영양 목표를 단정하지 않는다.

## 4. 핵심 화면

### 홈

- 안전 알림: 의료 처방/인슐린 조정 도구가 아님
- 음식 검색
- 선호 음식 레시피 바로 보기
- 한 끼 원칙 요약

### 음식 검색

필드:

- 음식명/동의어: 칼국수, 수제비, 김치전, 부추전, 두부부침 등
- 유형: 주식, 국수/면, 전/부침, 단백질, 채소, 간식, 음료
- 혈당 영향: 높음/중간/낮음
- 권장 등급: 자주 가능/조절 가능/가급적 피함/응급 또는 특별 상황
- 핵심 이유: 정제 탄수화물, 튀김/기름, 설탕, 나트륨, 단백질/섬유질 등
- 먹는 법: 분량, 같이 먹을 음식, 조리 팁

### 선호 음식 리디자인

- 칼국수: 면 양 줄이기, 국물 적게, 채소/버섯/단백질 늘리기, 밥/전과 같이 먹지 않기
- 수제비: 반죽 양 줄이기, 애호박/버섯/두부/계란 추가, 국물 나트륨 줄이기
- 김치전/부추전: 밀가루 반죽을 얇게, 채소 비율 높게, 기름 최소화, 작은 조각으로 분량 고정
- 두부부침: 비교적 유리하나 전분가루/양념장 설탕/기름 과다를 제한

### 장보기/준비 힌트

- 자주 쓸 채소, 단백질, 양념 목록
- 밀가루 음식의 반죽/분량 줄이는 방법
- 국물과 나트륨 줄이는 방법

## 6. 우선순위

1. 음식 분류와 검색 가능한 seed 데이터 작성
2. 선호 음식 6개 대체 레시피 상세화
3. GitHub Pages 공유 설정
4. 모바일 우선 UI 개선
5. 음식 데이터 확장

## 7. 필요한 추가 정보

- 정확한 키, 몸무게, 최근 공복/식후 혈당
- 당뇨 유형, 복용약, 인슐린 종류와 투여 시간. 용량 자체는 민감 정보로 앱에 저장하지 않아도 됨
- 신장 기능(eGFR), 단백뇨, 심혈관질환, 고혈압, 이상지질혈증 여부
- 저혈당 경험과 증상 인지 여부
- 하루 식사 시간, 간식, 야식, 음료 습관
- 씹기/삼킴/소화 문제
