# P09 Mom Diabetes Dashboard Agent Guide

## 역할

이 프로젝트는 인슐린 치료 중인 1960년생 여성 보호자를 위한 당뇨 식단 선택 대시보드다. 목표는 의료 처방이나 건강 데이터 추적을 대신하는 것이 아니라, 보호자가 어머니가 좋아하는 음식을 더 건강한 방식으로 바꾸고 필요할 때 음식 정보를 검색하도록 돕는 것이다.

## 사용자/대상자 초기 정보

- 대상자: 1960년생 여성
- 추정 신체 정보: 160cm, 75kg. 사용자가 정확한 키/몸무게를 나중에 제공 예정
- 최신 A1C: 11.4% (2026-07-01 사용자 제공)
- 치료 맥락: 인슐린 사용량이 많음
- 선호 음식: 칼국수, 수제비, 찌짐, 김치전, 부추전, 두부부침
- 주요 문제: 복잡한 혈당/운동/인슐린 기록을 지속하기 어렵고, 실제 식사 선택 순간에 쉬운 안내가 필요함

## 안전 원칙

- A1C 11.4%는 합병증 위험이 높은 수준으로 보고, 앱 내 문구는 반드시 의료진 상담을 권장한다.
- 인슐린 사용자는 식사량 감소, 운동, 탄수화물 제한 변화가 저혈당 위험을 높일 수 있다. 식단 조절 안내에는 저혈당 확인/대응 문구를 함께 둔다.
- 약물 용량, 인슐린 조정, 운동량 처방, 금식, 극단적 저탄수화물 식단을 권하지 않는다.
- 신장질환, 심혈관질환, 치아/삼킴 문제, 저혈당 무감지 여부가 확인되기 전까지 단백질, 칼륨, 나트륨 목표를 단정하지 않는다.
- 모든 건강 정보는 출처 URL과 확인일을 기록한다.

## 대시보드 범위

- 음식 검색: 음식명, 재료, 조리법, 탄수화물 위험도, 권장 빈도, 대체 팁
- 선호 음식 리디자인: 칼국수/수제비/전류를 완전 금지가 아니라 분량, 빈도, 조합, 조리법으로 조정
- 음식 검색: 음식명, 재료, 조리법, 탄수화물 부담, 권장 빈도, 대체 팁
- 레시피 안내: 칼국수, 수제비, 김치전, 부추전, 찌짐, 두부부침 등 선호 음식 중심
- 장보기/준비 힌트: 채소, 단백질, 양념, 조리법 중심
- 제외 범위: 인슐린 투여량 기록, 운동량 측정, 혈당 추세 분석 중심 대시보드

## 보고 형식

- 완료: 구현/문서화한 결과
- 진행 중: 남은 작업과 이유
- 차단: 사용자 입력 또는 외부 확인이 필요한 항목
- 결정 필요: 선택지와 추천안
- 다음 작업: 바로 실행 가능한 작업
