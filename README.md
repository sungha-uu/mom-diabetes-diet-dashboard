# P09 어머니 당뇨 식단 대시보드

인슐린 치료 중인 어머니의 식사 선택을 돕기 위한 보호자용 대시보드 프로젝트다. 혈당, 인슐린 투여량, 운동량을 체계적으로 측정하는 앱이 아니라, 좋아하는 한국 음식을 더 건강하게 바꿔 먹고 필요할 때 음식 정보를 검색하는 정적 안내 페이지다.

## 현재 기준 정보

- 대상자: 1960년생 여성
- 추정 체형: 160cm, 75kg (BMI 약 29.3, 정확한 정보 추후 갱신)
- A1C: 11.4% (2026-07-01 사용자 제공)
- 선호 음식: 칼국수, 수제비, 김치전, 부추전, 찌짐, 두부부침
- 치료 맥락: 인슐린 사용량이 많음

## 초기 산출물

- `index.html`: GitHub Pages용 정적 대시보드
- `docs/dashboard-plan.md`: 대시보드 구상 및 기능 범위
- `docs/food-and-recipe-guide.md`: 음식 분류와 선호 음식 대체 레시피 초안
- `data/food-taxonomy.seed.json`: 검색용 음식 데이터 초안

## 주의

이 프로젝트는 의료 처방, 인슐린 용량 조정, 운동량 처방, 혈당 추세 분석 도구가 아니다. 식단 변경, 저혈당 대응, 약물 조정은 반드시 주치의/당뇨교육자/영양사와 상의해야 한다.
